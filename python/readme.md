
## 🐍 Python Studies
Repositório central de estudos sobre o ecossistema Python. Aqui documento minha jornada desde os scripts fundamentais até o desenvolvimento de APIs e análise de dados.

## 🎯 Objetivos
Dominar a sintaxe e as PEPs (especialmente a PEP 8).
Explorar bibliotecas do ecossistema (Pandas, FastAPI, Requests).
Praticar lógica de programação e automação de tarefas.

## 📂 Organização
basics/: Tipos de dados, loops, condicionais e funções.
intermediate/: List Comprehensions, Decoradores, Iteradores e Geradores.
oop/: Classes, Mixins e métodos especiais (Dunder methods).
scripts/: Pequenas automações e utilitários úteis.
notebooks/: Experimentos realizados em Jupyter Notebooks.

## ⚙️ Configuração do Ambiente
Recomenda-se o uso de um ambiente virtual para rodar os projetos:
bash
# Criar ambiente virtual
python -m venv .venv

# Ativar (Linux/macOS)
source .venv/bin/activate

# Ativar (Windows)
.venv\Scripts\activate

# Instalar dependências (se houver)
pip install -r requirements.txt
Use o código com cuidado.
