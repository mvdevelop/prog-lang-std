
## 📑 Estudos C++
Repositório dedicado ao meu aprendizado da linguagem C++, cobrindo desde a sintaxe básica até conceitos avançados de Programação Orientada a Objetos (POO) e Estruturas de Dados.

## 🚀 Status do Aprendizado
Sintaxe Básica e Tipagem
Estruturas de Controle e Laços
Funções e Escopo
Ponteiros e Alocação Dinâmica
Programação Orientada a Objetos (POO)
Standard Template Library (STL)

## 📁 Estrutura do Repositório
O conteúdo está organizado por módulos temáticos:
01-fundamentos/: Variáveis, tipos de dados e operadores.
02-condicionais-e-loops/: if/else, switch, for, while.
03-funcoes/: Protótipos, passagem por referência vs valor.
04-arrays-e-strings/: Manipulação de vetores e a classe std::string.
projetos-praticos/: Pequenos programas desenvolvidos para fixação.

## 🛠️ Como Executar
Para compilar os arquivos deste repositório, você precisará de um compilador (GCC/G++ ou Clang).
bash
# Exemplo de compilação de um arquivo
g++ -o programa nome_do_arquivo.cpp

# Executando
./programa
Use o código com cuidado.
