
## 📘 TypeScript Studies
Este repositório é dedicado ao meu aprendizado e prática diária de TypeScript. Aqui documento a evolução dos meus estudos, desde conceitos básicos até padrões avançados e integrações.

## 🚀 Objetivo
O foco principal é dominar a tipagem estática do TypeScript para escrever códigos mais seguros, escaláveis e fáceis de manter, aplicando esses conceitos em ambientes Node.js e Front-end.

## 📂 Estrutura do Repositório
bash
.
├── 01-fundamentals/      # Tipos primitivos, interfaces e enums
├── 02-advanced-types/    # Generics, Utility Types e Decorators
├── 03-poo/               # Classes, Herança e Modificadores de acesso
├── 04-challenges/        # Resoluções de desafios (Exercism, LeetCode)
└── projects/             # Pequenas aplicações práticas
Use o código com cuidado.

## 🛠️ Tecnologias e Ferramentas
Linguagem: TypeScript
Runtime: Node.js / Bun
Linter/Formatter: ESLint & Prettier
Testes: Jest ou Vitest (em breve)
