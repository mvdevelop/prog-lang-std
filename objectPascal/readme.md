
## 🧬 Object Pascal Studies
Repositório dedicado ao estudo da linguagem Object Pascal, cobrindo desde a sintaxe básica até conceitos avançados de Programação Orientada a Objetos (POO).

## 🚀 Objetivo
Este projeto serve como um diário de aprendizado e guia de referência rápida para implementações em Pascal (Delphi/Free Pascal). O foco é consolidar conceitos de tipagem forte, gerenciamento de memória e estruturas de dados.

## 📂 Estrutura do Repositório
01-fundamentos/: Variáveis, tipos primitivos, operadores e estruturas de controle.
02-poo/: Classes, interfaces, herança, polimorfismo e encapsulamento.
03-generics/: Implementação de listas e coleções genéricas.
04-projetos/: Pequenas aplicações de console e desafios de lógica.

## 🛠️ Ferramentas Utilizadas
Compilador: Free Pascal (FPC) ou Delphi.
IDE: Lazarus, Delphi Community Edition ou VS Code (com extensão OmniPascal).

## 📝 Como Executar
Para compilar via terminal usando o FPC:
bash
fpc nome_do_arquivo.pas
./nome_do_arquivo
Use o código com cuidado.
