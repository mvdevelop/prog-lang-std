
# 🦀 Rust Studies

Repositório dedicado ao aprendizado da linguagem Rust, cobrindo desde conceitos fundamentais até o desenvolvimento de sistemas seguros e performáticos.

## 🎯 Objetivos
- Dominar os conceitos de **Ownership**, **Borrowing** e **Lifetimes**.
- Praticar a resolução de problemas com foco em segurança de memória.
- Construir ferramentas CLI e APIs utilizando o ecossistema Cargo.

## 📂 Estrutura do Projeto
- `01_basics/`: Variáveis, tipos primitivos e funções.
- `02_ownership/`: Práticas com o Borrow Checker.
- `03_structs_enums/`: Estruturas de dados e Pattern Matching.
- `04_collections/`: Uso de Vectors, Strings e HashMaps.
- `05_error_handling/`: Estratégias com `Result` e `Option`.
- `projects/`: Pequenas aplicações e ferramentas práticas.

## 🛠️ Requisitos
- [Rust & Cargo](https://rust-lang.org) instalados.
- Extensão `rust-analyzer` (recomendado para VS Code).

## 🚀 Como Executar
1. Clone o repositório:
   ```bash
   git clone https://github.com
   ```
2. Acesse o diretório do exercício desejado:
   ```bash
   cd 01_basics
   ```
3. Execute o código:
   ```bash
   cargo run
   ```
