
# 🍏 Smalltalk Studies

Repositório dedicado ao estudo de **Smalltalk**, explorando o paradigma de orientação a objetos puro e o ambiente interativo de desenvolvimento.

## 🎯 Objetivos
- Compreender a filosofia "Everything is an Object".
- Praticar o envio de mensagens (Unary, Binary e Keyword messages).
- Explorar o uso de Imagens e ambientes como Pharo ou Squeak.

## 📂 Estrutura do Projeto
- `01_syntax/`: Expressões básicas, blocos e variáveis.
- `02_classes/`: Definições de classes e hierarquias.
- `03_collections/`: Uso de OrderedCollection, Set e Dictionary.
- `04_metaprogramming/`: Reflexão e manipulação do sistema.
- `exercises/`: Desafios de lógica resolvidos em Smalltalk.

## 🛠️ Ferramentas
- **Ambiente:** [Pharo](https://pharo.org) ou [Cuis-Smalltalk](https://cuis.st).
- **Gerenciador:** Iceberg (para integração com Git).

## 🚀 Como Usar
Como Smalltalk é baseado em **imagens**, a melhor forma de explorar este repo é:
1. Abrir seu ambiente Smalltalk (ex: Pharo Launcher).
2. Importar os pacotes via **Iceberg** apontando para este diretório.
3. Explorar as classes via **System Browser**.
