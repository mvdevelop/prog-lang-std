
## 🎯 Kotlin Studies
Repositório central para meus estudos, anotações e experimentos com a linguagem Kotlin. O objetivo aqui é explorar desde a sintaxe idiomática até o desenvolvimento multiplataforma.

## 🚀 Por que Kotlin?
Kotlin é moderno, conciso e focado em segurança (especialmente o famoso Null Safety). Meus estudos abrangem o uso da linguagem para:
Desenvolvimento Android (Nativo)
Backend (Spring Boot / Ktor)
Scripting e Automação

## 🛠️ Ferramentas Utilizadas
Linguagem: Kotlin (JVM)
Build System: Gradle (Kotlin DSL)
IDE Recomendada: IntelliJ IDEA / Android Studio
Testes: JUnit 5 e MockK

## ⚙️ Como rodar os projetos
Como este diretório utiliza o Gradle, você pode rodar os exemplos via terminal:
