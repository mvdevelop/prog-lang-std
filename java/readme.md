
# ☕ Java Mastery: Estudos e Práticas

Este repositório é dedicado ao meu aprendizado contínuo em Java, cobrindo desde a base acadêmica até padrões de projeto avançados e arquitetura de software.

## 📁 Estrutura do Repositório

O projeto está dividido em três pilares principais:

### 1. 🎓 [Graduação Java POO](./graduacao-java-poo)
Fundamentos da Programação Orientada a Objetos aplicados em ambiente acadêmico.
* Abstração, Encapsulamento, Herança e Polimorfismo.
* Estruturas de dados básicas e lógica de programação com Java.

### 2. 🏛️ [Design Patterns](./design-patterns)
Implementação de soluções comuns para problemas recorrentes no desenvolvimento de software.
* **Criacionais:** (ex: Singleton, Factory, Builder)
* **Estruturais:** (ex: Adapter, Composite, Proxy)
* **Comportamentais:** (ex: Strategy, Observer, State)

### 3. 🛡️ [SOLID](./solid)
Projetos focados na aplicação dos cinco princípios da programação orientada a objetos para criar sistemas fáceis de manter e estender:
* **S** - Single Responsibility
* **O** - Open/Closed
* **L** - Liskov Substitution
* **I** - Interface Segregation
* **D** - Dependency Inversion
