
# 🚀 C Lang Studies

Repositório dedicado ao aprendizado e prática da linguagem **C**. Aqui documento minha evolução, desde a sintaxe básica até conceitos avançados de manipulação de memória e estruturas de dados.

## 📂 Estrutura do Repositório

O conteúdo está organizado por tópicos para facilitar a navegação:

- `basics/`: Variáveis, tipos de dados, operadores e estruturas de controle.
- `functions/`: Protótipos, escopo e recursividade.
- `pointers/`: Aritmética de ponteiros, referências e alocação dinâmica.
- `data-structures/`: Listas ligadas, pilhas, filas e árvores.
- `exercises/`: Desafios de lógica e problemas resolvidos (Beecrowd, LeetCode, etc).
