
## 🎯 Estudos C# & .NET
Este repositório contém meus exercícios, anotações e pequenos projetos desenvolvidos durante o aprendizado de C# e da plataforma .NET.

## 🛤️ Trilha de Aprendizado
Lógica de Programação com C#
Orientação a Objetos (Classes, Herança, Interface)
Manipulação de Exceções
LINQ (Language Integrated Query)
Entity Framework Core (Banco de Dados)
ASP.NET Core (APIs)

## 🗂️ Organização
Os estudos estão divididos por projetos de console, seguindo a estrutura de Solutions:
Fundamentos/: Tipos primitivos, condicionais e repetição.
OrientacaoObjetos/: Exemplos de encapsulamento, polimorfismo e abstração.
Colecoes/: Listas, Dicionários e arrays.
Projetos/: Pequenos sistemas (ex: Calculadora, Sistema Bancário, RPG de Console).

## 💻 Como Rodar os Projetos
Você precisará do .NET SDK (6.0, 7.0 ou 8.0) instalado.
bash
# Entrar na pasta do projeto desejado
cd PastaDoProjeto

# Restaurar dependências
dotnet restore

# Executar o projeto
dotnet run
Use o código com cuidado.
